Installation from Package
------------

Installation from Git Source
------------

1. Check build-requirements:

 ```
 * autoconf
 * automake
 * libglib2.0-dev (glib2)          >= 2.48.0
 * libxml2-utils (libxml2)
 * pkg-config (pkgconfig)
 ```

3. Build and install system-wide:

 ```
 ./autogen.sh
 make
 sudo make install
 ```

 > **Note:**
 >
 >   * Default prefix is `/usr`.
